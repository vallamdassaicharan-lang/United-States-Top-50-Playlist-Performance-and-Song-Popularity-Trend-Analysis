# Top 50 Songs Analysis

This project has been split into a small Flask web app so the analysis page can be run locally with a cleaner file structure.

## Run locally

1. Create and activate a virtual environment
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the app:
   ```bash
   python app.py
   ```
4. Open http://127.0.0.1:5000/
